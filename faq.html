1. Backup e Restauração de Arquivos

Consigo fazer um backup de arquivo em um Pendrive? +

Verifique o funcionamento do dispositivo que você está tentando usar para fazer o backup dos arquivos. Faça testes com outros programas de gravação. Se houver problemas, solicite auxílio do seu Técnico em Informática.

Como eu faço para restaurar os arquivos do Neuromap? +

Utilizando as versões do Neuromap com Windows XP ou superior:

  1. Entre no programa Neuromap;
  2. Na tela principal, selecione no menu Arquivo -> Transferir -> Exames;
  3. No quadro que aparece, selecione a origem dos arquivos (CD, DVD, Pendrive…) e marque os exames que serão copiados;
  4. Em seguida, no destino da cópia, selecione a pasta de exames do Neuromap;
  5. Pressione o botão Transferir.

Utilizando as versões do Neuromap com Windows 98:

  1. Entre no Windows Explorer;
  2. Selecione a origem dos arquivos (CD, DVD, Pendrive…);
  3. Marque os exames que serão copiados;
  4. Abra a pasta Neurowin/exames;
  5. Arraste os arquivos para a pasta;
  6. Remova o atributo "somente leitura" dos arquivos;
  7. Selecione a origem dos arquivos (CD, DVD, Pendrive…);
  8. Marque os arquivos ficha.dat e analise.dat;
  9. Abra a pasta Neuromap/Dados;
  10. Arraste os arquivos para a pasta;
  11. Remova o atributo "somente leitura" dos arquivos.

Caso não consiga realizar a restauração de arquivos, solicite auxílio do seu Técnico em Informática.

Como eu faço um backup de Arquivo em um Pendrive? +

Utilizando as versões do Neuromap com Windows XP ou superior:

  1. Entre no programa Neuromap;
  2. Na tela principal, selecione no menu Arquivo -> Transferir -> Exames;
  3. No quadro que aparece, marque os exames que serão copiados;
  4. Em seguida, no destino da cópia, digite o caminho de destino ou pressione Procurar -> Meu Computador, escolha a opção do Pendrive;
  5. Pressione o botão Transferir.

Utilizando as versões do Neuromap com Windows 98:

  1. Entre no Windows Explorer;
  2. Abra a pasta Neurowin/Exames;
  3. Marque os exames que serão copiados;
  4. Arraste os arquivos para o Pendrive;
  5. Abra a pasta Neurowin/Dados;
  6. Marque os arquivos ficha.dat e analise.dat;
  7. Arraste os arquivos para o Pendrive.

Caso não consiga realizar o backup, solicite auxílio do seu Técnico em Informática.

Em quais tipos de mídia posso fazer Backup de exames? +

O programa Neuromap pode fazer backup direto em mídias do tipo R/W (CD R/W, DVD R/W, Pendrive, etc). O Neuromap não faz backup direto em mídias do tipo R (somente leitura); neste caso, utilize os programas de gravação que acompanham o gravador.

2. Configuração de Rede

O Neuromap pode ser utilizado em rede com um computador gravando exames e outro fazendo laudos. Para funcionar corretamente é necessário que a rede esteja operando adequadamente. Para lhe auxiliar com a instalação e configuração da rede, entre em contato com o seu Técnico de Informática.

Consigo abrir exames para fazer análise em uma rede? +

Verifique no ambiente de rede do computador se é possível o acesso às unidades de rede e se o computador em que estão os exames está disponível. Se a rede não estiver com funcionamento normal ou não for possível acessar o computador de exames, solicite auxílio do técnico em informática responsável pela rede.

Como eu faço para configurar o Neuromap para fazer análise em uma rede? +

Siga os passos abaixo:

  1. Instale o programa Neuromap no computador de gravação de exames;
  2. Para definir e configurar da melhor forma possível o compartilhamento e consequentemente o caminho para acessá-lo, chame um técnico de sua confiança. No computador de análise, peça para o técnico fazer o mapeamento do compartilhamento do computador de exames;
  3. No computador de análise, instale o programa Neuromap;
  4. No computador de análise, entre no programa Neuromap. Selecione no menu a opção: Configurações -> Programa -> Pastas, ou Configurações -> Arquivos/Pastas. Troque a letra da unidade C: pela unidade mapeada nas opções referentes a 'Exames', 'Dados', 'Telas' e 'Laudos'.

3. Eletrodos

Eletrodos são considerados materiais de consumo. A duração destes materiais depende da frequência de uso e cuidados com a sua conservação. Após apresentarem problemas, devem ser prontamente substituídos e descartados. Mantenha sempre uma reserva para substituição oportuna.

Como faço para conservar os eletrodos? +

Após o uso, lave os eletrodos em água corrente para retirar a pasta impregnada, esfregando-os com uma escova macia. Seque-os e os mantenha apoiados na maca ou cadeira utilizada para fazer o exame. Nunca deixe os eletrodos de molho na água de um dia para o outro. Isto diminui consideravelmente a vida útil do componente.

Onde devo montar o eletrodo de referência? +

Recomenda-se a posição Fpz para o eletrodo de referência único (na região frontal, entre os eletrodos Fp1 e Fp2). Outras posições de referência podem ser utilizadas a critério do médico, como na região mandibular (um de cada lado) – como o registro é sempre feito com a referência, se uma falhar a outra "segura" o registro.

Como faço para verificar se o eletrodo está bom? +

No programa Neuromap faça o teste de impedância após a colocação dos eletrodos na cabeça do paciente. Se os valores de impedância de um eletrodo ficarem sempre altos (após a remontagem), é necessário fazer a substituição desse eletrodo.

Utilize o teste de continuidade de eletrodos do programa Neuromap. Para fazer este teste, veja a orientação no manual do usuário.

Os eletrodos não podem apresentar fios duros próximo aos conectores e conchas. Também não podem apresentar a capa do fio rasgada ou mordida com uma parte do condutor exposta.

Qual é o tipo de eletrodo que posso utilizar com o Neuromap? +

Com o equipamento Neuromap podem ser utilizados os eletrodos do tipo de estanho, ouro, prata e cloridrato de prata (prata clorada). Mas atenção: não misture eletrodos de tipos diferentes, pois o conjunto de materiais distintos gera artefatos na captação do exame.

Quando deve ser feita a troca do eletrodo? +

O eletrodo deve ser trocado individualmente sempre que apresentar problemas de mau contato ou falta de contato. Em no máximo seis meses, faça uma troca completa de todo o conjunto de eletrodos.

4. Equipamento não inicia a captação

O sistema não inicia a captação de sinal e aparece a mensagem "Erro de inicialização do Sistema USB"? +

Faça as seguintes verificações:

  1. Verifique se o cabo USB está conectado corretamente ao computador e ao equipamento;
  2. Troque a entrada (Porta) do cabo USB no computador;
  3. Substitua o cabo USB por outro para testes;
  4. Verifique se o programa foi instalado corretamente no computador. O sistema possui um 'Driver USB' que efetua a comunicação entre o equipamento Neuromap e o PC. Se o Driver não estiver ativo, o equipamento não inicia a captação;
  5. Tente instalar o equipamento em outro computador;
  6. Se o problema persistir, entre em contato com o suporte técnico da Neurotec.

5. Interferências na captação do sinal

Por que o equipamento começou a captar o exame com muita interferência de 60Hz no mesmo local onde sempre fez exames? +
  • Solicite ao eletricista responsável uma verificação das condições da tomada, da fiação e do aterramento do sistema (certifique-se de que a impedância está abaixo de 5 Ohms);
  • Verifique se não há alguma reforma recente no local que possa interferir na rede elétrica e no aterramento;
  • Verifique se algum cabo do computador, estabilizador, filtro de linha ou outro equipamento elétrico não rompeu o fio de aterramento;
  • Troque o eletrodo de aterramento do paciente (por exemplo, FPZ).
Após a mudança de local do equipamento, o exame é captado sempre com muita interferência? +
  • Solicite ao eletricista responsável uma verificação das condições da tomada, da fiação e do aterramento do sistema (verifique se ele existe no novo local e se a impedância está abaixo de 5 Ohms);
  • Verifique se existem outros aparelhos próximos como ressonância magnética, aparelhos de Raios-X, consultórios dentários, motores, elevadores, etc. Caso existam, este ambiente pode ser inadequado para exames de neurofisiologia clínica.
Como faço para verificar se o equipamento está captando sinal corretamente? +

O Neuromap possui vários recursos para verificação do seu funcionamento correto, que são: o Teste de Calibração Individual, o Teste de Calibração Conjunta, o Teste de Continuidade dos Eletrodos e o Teste de Impedâncias. Execute estes testes conforme orientado no manual do usuário para confirmar o funcionamento ideal.

Alguns canais do exame gravado ficam isoelétricos em montagens bipolares? +

Verifique se há formação de ponte salina entre os eletrodos (ligação de pasta entre eletrodos muito próximos). Há maior probabilidade de que isto aconteça se não for seguido rigorosamente um padrão de montagem, como o Sistema Internacional 10-20.

Estou captando sinal com interferência ou isoelétrico em todos os canais? +
  • Caso a interferência esteja ocorrendo em TODOS os canais, corrija primeiro os eletrodos de referência, substituindo-os se necessário;
  • Verifique se o paciente está encostando alguma parte do corpo na parede, no chão ou em algum objeto metálico (maca). Em caso afirmativo, peça para desencostar;
  • Verifique se a temperatura da sala de exames está alta. O calor excessivo afeta diretamente a captação. Se o paciente estiver com sudorese, poderão ocorrer artefatos de onda lenta de grande amplitude;
  • Verifique se os eletrodos são todos do mesmo tipo. Não utilize, ao mesmo tempo, eletrodos de materiais diferentes ou de fabricantes diversos;
  • Verifique a impedância de contato dos eletrodos. Caso haja canais com impedância alta, limpe melhor a superfície da pele em contato com a pasta condutora;
  • Substitua os eletrodos dos canais em que a impedância não diminui após a leve abrasão da pele;
  • Evite usar carpetes e tapetes na sala de exames, pois eles acumulam carga estática, gerando ruído;
  • A qualidade da pasta condutora é fundamental. Mesmo usando a mesma marca, pode haver variação de lote. Considere realizar um teste com pasta de outro lote.

6. Laudo Remoto

Como eu faço para laudar exames remotamente, via Internet? +

Existem diversas maneiras de fazer um laudo remotamente. A mais comum é criar uma rede externa segura com acesso pela Internet. Para estruturar essa conexão, solicite o auxílio do seu Técnico de Informática. Após criar a rede externa, configure o sistema como uma rede comum local.

Outro modo é realizando a transferência manual dos arquivos de exame via internet para o computador de destino onde o laudo será elaborado.

7. Laudos no Word

Como faço para personalizar um Laudo no Word? +

Se desejar fazer modificações para personalizar algum modelo de laudo padrão, siga estes passos:

  1. Abra a pasta Neuromap/modelos/EEG ou EEGQ dentro do drive local (C:);
  2. Clique sobre o ícone do modelo desejado com o botão direito do mouse e selecione a opção Abrir (não dê duplo clique direto para não criar um documento novo baseado nele);
  3. Faça as modificações desejadas (atenção: não altere a grafia das palavras que estão entre o sinal #);
  4. Selecione no menu do Word a opção Salvar;
  5. Feche o arquivo.
É possível fazer o laudo de um exame gravado em CD? +

Se desejar laudar exames que estão armazenados em um CD, DVD ou Pendrive, copie antes todos os arquivos correspondentes (com o nome do paciente) para a pasta local 'Exames'. Verifique se eles foram copiados com o atributo de 'Somente Leitura' ativo; se sim, desmarque essa opção nas propriedades do arquivo.

Por que não consigo fazer o laudo em outro computador ligado em rede? +

Se estiver laudando exames através de uma rede local, certifique-se de que o computador servidor onde os arquivos estão gravados está ligado e disponível para acesso. Caso note falhas de compartilhamento, acione o técnico responsável pela rede.

Por que o laudo não fica preenchido corretamente? +

Se alguma informação automatizada (macro) não estiver preenchendo, abra o arquivo original do modelo dentro do diretório C:\Neuromap\modelos\EEG (clicando com o botão direito e selecionando Abrir). Verifique se as tags foram alteradas acidentalmente com espaços, acentos ou cedilhas onde não deveria. Corrija a grafia, salve e feche.

Por que o laudo no Word não inicia? +

A integração e montagem do documento ocorrem por meio de uma Macro do Word que pode estar bloqueada pelas diretivas de segurança do Office. Para habilitar:

  • Versões do Word anteriores ao Word 2007: Acesse o menu Ferramentas -> Macros -> Segurança, e defina o nível como 'Baixo'.
  • Versão do Word 2007 ou superior: Pressione o Botão Office / Arquivo -> Opções do Word -> Central de Confiabilidade -> Configuração da Central de Confiabilidade -> Configurações de Macro -> Habilitar todas as macros.

Atenção: Softwares maliciosos (vírus) também podem corromper as funções do Word. Mantenha um antivírus ativo e atualizado. Se o Word apresentar comportamento instável geral, solicite a reinstalação do pacote Office ao seu técnico.

8. Sistemas com Windows 98

Como eu faço para configurar o Neuromap para fazer análise em uma rede se o computador de exames está com Windows 98? +
  1. Configure o compartilhamento físico da pasta no Windows 98 com o suporte de um técnico. No computador moderno de análise, realize o mapeamento dessa unidade de rede;
  2. Instale o programa Neuromap no computador de análise;
  3. Pressione o botão Iniciar do Windows e selecione a opção Executar;
  4. Na caixa de diálogo aberta digite Neuromap.ini e pressione OK;
  5. No arquivo de texto aberto via Bloco de Notas, localize e modifique as linhas referentes a DAT, EXM, DADOS e EXAMES nas seções [PATHS] e [DIRETÓRIOS]. Substitua a letra padrão 'C' pela respectiva letra da unidade de rede mapeada;
  6. Feche o Bloco de Notas salvando as alterações.
Meu computador funciona com Windows 98 e não consigo instalar uma impressora nova para imprimir exames? +

Impressoras modernas geralmente não possuem drivers compatíveis com o Windows 98. A solução é conectar esse computador em rede local. Em outra máquina mais moderna da rede, instale o Neuromap, configure o acesso aos exames remotamente e realize as impressões por ela.

Fiz um upgrade no Windows 98 e o meu equipamento parou de registrar sinal? +

As versões de software antigas dedicadas ao Windows 98 não possuem compatibilidade de arquitetura para gravar sinais em sistemas operacionais como Windows XP, Vista, Windows 7 ou superiores. Entre em contato com o suporte da Neurotec para consultar os pacotes de atualização para o seu modelo.

O sistema inicia a captação de sinal, mas o mesmo está sempre reto? +

Se a linha iniciar completamente reta (isoelétrica), verifique se os LEDs indicadores do equipamento físico estão acesos. Se estiverem apagados, verifique as conexões de energia, a chave liga/desliga ou cabos soltos. Se continuarem apagados mesmo com alimentação correta, contate nosso suporte técnico.

O sistema não inicia a captação de sinal e não corre sinal na tela? +

Se a tela de captação ficar estática ou em branco, solicite ao técnico de informática que verifique a BIOS do computador na seção "PCI/PNP CONFIGURATION". Os pedidos de interrupção IRQ 3 e 5 devem estar devidamente reservados para ISA, e a porta serial física COM2 deve ser desabilitada.

O programa não abre a ficha de seleção de pacientes? +

Caso o sistema trave ao tentar acessar o cadastro de pacientes, localize o diretório raiz onde o sistema está instalado (por exemplo, C:\Neurowin\Dados), remova manualmente os arquivos indexadores corrompidos Ficha.Dat e Analise.Dat, e reinicie o sistema para que ele recrie tabelas limpas.
